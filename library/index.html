Color Utils

Этот файл демонстрирует использование утилит для работы с цветами.

  • hexToRgb: Преобразует шестнадцатеричный цвет в RGB.
  • rgbToHex: Преобразует RGB цвет в шестнадцатеричный формат.
  • randomColor: Генерирует случайный цвет в шестнадцатеричном формате.
  • strr: Переворачивает строку.
